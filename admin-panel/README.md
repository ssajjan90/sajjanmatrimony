# Sajjan Matrimony - Admin Panel

React (Vite) web admin dashboard.

## Prerequisites

- Node.js 18+
- npm or yarn

## Running

```bash
npm install
npm run dev
```

Admin panel: http://localhost:5173

## Pages

| Page              | Route      |
|-------------------|------------|
| Admin Login       | /login     |
| Dashboard         | /dashboard |
| Profile Approvals | /approvals |
| User Management   | /users     |
| Reports           | /reports   |
